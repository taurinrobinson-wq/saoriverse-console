## 💡 ROOT CAUSE ANALYSIS

### Why the Imbalance?

The imbalance stems from the **CSV source glyph distribution**:

1. **Original CSV** (emotional_os/glyphs/glyph_lexicon_rows.csv):
   - Contains high concentration of Gates 3 & 7 glyphs
   - Sparse coverage of Gates 1, 4, 11, 12

2. **Factorial Expansion** process:
   - Generated combinations by pairing existing glyphs
   - High-count gates produce more combinations
   - Low-count gates produce fewer combinations
   - Result: Initial imbalance is **amplified** not corrected

3. **Validation/Pruning** process:
   - Removed 2,126 "broken" glyphs uniformly
   - Did NOT rebalance gate distribution
   - Maintained proportional imbalance

