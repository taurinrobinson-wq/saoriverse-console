## Column Quick Reference

### Navigation
- **Core Identity:** glyph_name, NPC Giver, Category, Theme
- **Story Progression:** unlock_condition, location, Storyline
- **Fusion Progression:** fusion_group, required_glyphs, boss_gate, moral_choice
- **Emotional Integration:** semantic_tags, remnants_effects, stat_rewards
- **Flags & Gates:** flags_set, flags_required, required_stats, required_items

### Common Queries
```sql
-- Find glyphs by NPC
WHERE npc_source = 'Ravi & Nima'

-- Find fusion components
WHERE fusion_group = 'triglyph'

-- Find glyphs with specific semantic tag
WHERE semantic_tags LIKE '%grief%'

-- Find glyphs affecting empathy trait
WHERE remnants_effects LIKE '%empathy%'

-- Find glyphs locked by specific gate
WHERE unlock_condition = 'after_kaelen_confession'
```
