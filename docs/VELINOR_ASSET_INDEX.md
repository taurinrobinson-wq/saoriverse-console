### Import Workflow

```powershell
# 1. In Unity Project, click Assets menu
# 2. Select "Import Package" → "Custom Package"
# 3. Navigate to D:\Velinor\Asset Store-5.x\[Creator]/[Category]/[Package]
# 4. Select desired components (review dependencies)
# 5. Click "Import"
```

**Pro Tips:**
- Import one package at a time to isolate dependency issues
- Create a temporary test scene to validate imports before committing
- Check console for shader/material compatibility warnings
- Note: Tree packs may include multiple mesh variations—select only needed LODs
